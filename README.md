# ReviewMyResume (Streamlit)
A website that reviews your resume and gives you relevant feedback.

## Quickstart (VS Code)

1. **Open folder**
   - In VS Code: *File → Open Folder…* and select this project.

2. **Create a virtual environment (recommended)**
   ```bash
   python -m venv .venv
   # Activate
   .venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run**
   ```bash
   streamlit run streamlit_app.py
   ```

5. **Open the website**
   - Your browser should open automatically. If not, copy the provided URL into your browser.
